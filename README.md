## Getting Started

To run locally must run backend and frontend seperately each in a different powershell tab

npm - backend

```bash
git clone https://github.com/georgge293/MuscleUP-workout-planner.git
cd backend
npm install
npm run dev
```

npm - frontend

```bash
cd frontend
npm install
npm start
```
